
# Taggme — Plataforma Profissional Completa

## Como usar

### 🔁 Rodar localmente

1. Rode o backend:
```bash
cd backend
npm install
npm run dev
```

2. Rode o frontend:
```bash
cd frontend
npm install
npm run dev
```

Acesse: http://localhost:5173/feed

### 🚀 Deploy

- Frontend: Vercel
- Backend: Railway
- Banco: PostgreSQL

Veja `.env.example` no backend para configurar conexão.
